# ⚡ BrainBurst — Quiz Application

## 🔗 Live Demo
[Brain Burst Quiz App](https://https://bburst.vercel.app)

## ✨ Features
- User Register / Login
- Category wise Quiz (HTML, CSS, JS, React)
- 15 Second Timer
- Score & Result Page
- Leaderboard with Medal System
- Admin Dashboard — Add / Edit / Delete Quiz

## 🛠️ Tech Stack
React.js, Firebase Firestore, Tailwind CSS, React Router
